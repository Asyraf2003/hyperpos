# Active Error Log

Folder ini hanya untuk error log yang masih aktif atau sedang diremediasi.

Dokumen yang sudah selesai harus dipindahkan ke:

- `docs/99_archive/04_lifecycle/error_log/`

Rule:

- Jangan simpan issue closed di folder ini.
- Jangan membuat ulang investigasi lama tanpa bug evidence baru.
- Jika issue sudah fixed, verified, atau reclassified sebagai data/setup-specific, pindahkan ke archive.
- README ini boleh tetap tinggal sebagai penanda active workspace.
