# Archived Error Logs

Folder ini berisi error log lifecycle yang sudah closed, verified, reclassified, atau tidak lagi aktif.

Dokumen di sini adalah catatan historis, bukan active work queue.

Rule:

- Jangan lanjutkan pekerjaan langsung dari archive tanpa membuka issue/slice baru.
- Jangan pindahkan kembali ke active lifecycle kecuali ada bug evidence baru.
- Gunakan archive sebagai reference/proof, bukan sebagai daftar kerja harian.
