# P2 - Terminal Command Delivery

## Tujuan
Menyesuaikan delivery implementasi dengan preferensi user yang ingin command siap copy-paste.

## Aturan
- Bila cocok, berikan implementasi dalam bentuk terminal commands.
- Jika command terlalu panjang atau berisiko salah paste, pecah menjadi beberapa batch.
- Setiap batch harus punya tujuan jelas.
